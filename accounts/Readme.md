# Accounts
Es la aplicación encargada de gestionar los usuarios dentro de
Itzamná, donde se extiende la classe UserManager debido a los nuevos
campos que son agregados en el modelo User 
 
 
## Campos
   - first_name (Requerido)
   - last_name (Requerido)
   - username  (Requerido)
   - email  (Requerido)
   - course 
   - gender
